# Research inbox

Drop research documents here — papers (PDF), reports, notes, datasets, links files — anything that
should feed the channel-space exploration. No naming convention required; if you want a document
tied to a specific candidate channel, prefix it (e.g. `water--wri-aqueduct.pdf`, `food--ifpri-impact.md`).

**What happens to drops:** each new document gets read, summarized into
`../findings/` with an assessment of which candidate channel(s) it informs and whether it changes a
feasibility grade in `../channel-space-map.md`. The inbox itself stays raw — originals are never
edited or deleted.

Heavy binaries are fine here: this lane never merges wholesale into trunk, so inbox contents stay
lane-local. A `links.md` file works too if you'd rather paste URLs than files.
